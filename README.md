# GourdDollSaveGrandpa

###cd到project-g文件夹再执行下面的操作

## Build Setup

``` bash
# install dependencies
npm install

# install Vuex
npm install Vuex

# install vue-router
npm install vue-router

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
